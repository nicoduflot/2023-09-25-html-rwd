Les media

L'audio

Pour l'audio hébergé sur le site

Il existe la balise <audio>

Il est possible de donner des source de fichiers différentes : le navigateur va utiliser le premier format qu'il sera capable de lire.

L'audio hébérgé sur un site tier

Pour les media audio hébergés sur des sites tiers, il s'agit en fait de pages contenants un lecteur et la ressource audio qui sont appelés dans une iframe.

Grâce à ça, le lecteur peut être fait en HTML - CSS, les contrôles affichés sont généralement gérés avec du JavaScript

Youtube impose un format da l'iframe, il faudra modifier le CSS voir créer un élément enveloppeur afin de rendre l'iframe adaptative

Daily motion apporte une solution adaptative inclue, avec un élément conteneur et une iframe appliquant du style CSS enbalise afin d'assurer l'adaptabilité à la largeur de l'affichage (device-width)

Les iframes

Les iframes sont des cadres dans lesquels on appelle d'autres pages. Il faut que ces page soient autorisées par le serveur qui les héberge d'être partagées par un autre site.

Les iframe permette en réalite d'appeler quasiment tout type de fichier dans le cadre.
